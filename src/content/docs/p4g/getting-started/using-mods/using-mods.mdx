## Installing Mods

To install mods,

- If on Windows,
  - If modding the Steam version, follow [this guide](https://gamebanana.com/tuts/15696).
  - If modding the Microsoft Store/Game Pass version, follow [this guide](https://gamebanana.com/tuts/17171), replacing references to P3R with P4G.
- If on Steam Deck, follow [this guide](https://gamebanana.com/tuts/17474).
